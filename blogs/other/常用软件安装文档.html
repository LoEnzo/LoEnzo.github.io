常用软件安装文档

Enzo 2020-07-09 tools

有时候电脑太卡，需要重装系统，自己喜欢的软件免不了各种搜索，重新安装。这里进行一个汇总，方面以后进行查找。

# 编程相关：

  • JetBrains IDEA

  • VScode

  • JDK8

  • Maven

  • Mysql

  • Redis

  • Git

  • Node.js

  • Yarn

  • BCompare

  • JD-gui：查看jar包

  • Navicat Premuim

  • TortoiseGit

  • [XMLSpy2013：查看xsd，xml]

  • Postman

# 文档编辑器：

  • Typora：Markdown文档编辑器

  • Notepad++

# windows常用软件：

  • BandZip: 压缩软件
  • QTTabBar：Windows资管管理器插件，先下载1038，再下载1040补丁更新即可获取最新版

# 其他：

  • Everything：磁盘搜索软件

  • Utools：快捷启动软件

  • Windows Terminal: windows应用商店下载

  • QuickLook：快速预览

  • Snipaste：截屏贴图软件，Windows商店也可以下载

  • TrafficeMonitor：系统托盘图片，显示网速等参数信息
