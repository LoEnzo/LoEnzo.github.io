常用软件安汇总

Enzo 7/9/2020 tools

有时候电脑太卡，需要重装系统，自己喜欢的软件免不了各种搜索，重新安装。这里进行一个汇总，方面以后进行查找。

# 编程相关：

  • JetBrains IDEA (opens new window)

  • VScode (opens new window)

  • JDK (opens new window)

  • Maven (opens new window)

  • Mysql (opens new window)

  • Redis (opens new window)

  • Go (opens new window)

  • Git (opens new window)

  • Node.js (opens new window)

  • Yarn (opens new window)

  • [BCompare]，class反编译插件,选择对应版本，搜索class，下载，导入，重启 (opens new window)

    过期报错，删除C:\Users\用户名\AppData\Roaming\Scooter Software

  • JD-gui：查看jar包 (opens new window)

  • [Navicat Premuim：数据库连接工具]

  • TortoiseGit (opens new window)

  • [XMLSpy2013：查看xsd，xml]

  • Postman (opens new window)

  • Fiddler：抓包工具

# 文档编辑器：

  • Typora：Markdown文档编辑器 (opens new window)
  • Notepad++ (opens new window)
  • Sublime Text 3 (opens new window)

# windows常用软件：

  • BandZip: 压缩软件 (opens new window)
  • QTTabBar：Windows资管管理器插件 (opens new window)，使用参考，评论含中文文件下载 (opens new window)，汉化文件下载 (opens new window)

# Linux常用软件：

  • fish shell：终端命令自动补全，提示，校验等，help查看命令文档 (opens new window)

# 其他：

  • Everything：磁盘搜索软件 (opens new window)

  • Utools：快捷启动软件 (opens new window)

  • Windows Terminal：终端，windows应用商店下载

  • QuickLook：空格快速预览文件 (opens new window)

  • Snipaste：截屏贴图软件，Windows商店也可以下载 (opens new window)

  • TrafficeMonitor：系统托盘图片，显示网速等参数信息

  • 向日葵远程控制
